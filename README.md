## 📖 Conceitos Abordados

### Fila Circular
Estrutura FIFO (First In, First Out) implementada com array circular usando o operador `%` para evitar desperdício de espaço.

### Pilha
Estrutura LIFO (Last In, First Out), onde o último elemento inserido é o primeiro a ser removido.

### Lista Encadeada
Estrutura dinâmica onde cada nó aponta para o próximo, permitindo inserção e remoção eficientes.

### Árvore Binária de Busca
Estrutura hierárquica onde cada nó tem no máximo dois filhos, com a propriedade de que filhos à esquerda são menores e à direita são maiores.
